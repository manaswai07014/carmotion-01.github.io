## Engine Evolution Tree

```
Mercedes-AMG Engine Family
├── M159 6.2L Naturally Aspirated V8 (SLS AMG, SLS GT3, GT3)
│   └── 2010-2021：自然吸氣旗艦，571-631 hp
│
├── M178 4.0L Twin-Turbo V8 "Hot Inside V" (AMG GT family)
│   ├── 2014-2022 (C190)：RWD，7-speed DCT，585-730 hp
│   └── 2023+ (C192)：RWD/AWD，9-speed，585-816 hp
│
└── M139 + Hybrid (New Generation)
    ├── M139 2.0L Turbo I4 + EQ Boost (Mild Hybrid)
    └── F1-Derived PU106B Hybrid (Mercedes-AMG One)
        ├── 1.6L E-Turbo V6 (566 hp)
        └── + 4x Electric Motors (483 hp total) = 1,049 hp
```

## Key Architecture Patterns

```
Platform Generation:
├── SLS Platform (R190) → 鷗翼、6.2L V8自然吸氣、2010-2014
├── AMG GT Platform (C190) → 4.0L V8渦輪增壓、2014-2022
└── MSA Platform (C192/R232) → 第二代AMG GT + SL、2023+

Drivetrain Evolution:
├── RWD Only (早期) → SLS、AMG GT (C190)
├── RWD + AWD (4MATIC+) (中期) → AMG GT (C192)
└── AWD + Hybrid (旗艦) → AMG One PHEV、SL 63

Body Style:
├── 鷗翼跑車 → SLS AMG
├── 雙門coupé → AMG GT、AMG One
├── 軟頂敞篷 → SL 63 R232
└── 賽道專用 → GT3、GT4、Track Series、Black Series
```
